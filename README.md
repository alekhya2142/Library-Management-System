# Library-Management-System
## 📌 Overview
This is a simple **Library Management System** implemented in C++.  
It allows users to manage a collection of books using **file storage**.  
Core operations include:
- Adding new books
- Displaying all books
- Searching for books by title
- Deleting books by ISBN

---

## ⚙️ Features
- **Add Book**: Enter Title, Author, and ISBN, which are saved in `library.txt`.  
- **Display Books**: Shows all books in a clean format (Title, Author, ISBN).  
- **Search Book**: Finds books by title (supports partial matches).  
- **Delete Book**: Removes a book by its ISBN safely.  
- **Persistent Storage**: All data is stored in a text file, so it remains after program exit.
